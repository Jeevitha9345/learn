Expense Tracker

Overview
Expense Tracker is a Java console application that helps users manage personal expenses. Users can add expenses, view them, calculate the total, and find the highest expense. This project uses Java OOP concepts and ArrayList to store data.

Features
Add an expense with name and amount
View all expenses
Calculate total expenses
Find the highest expense
Input validation (prevents zero or negative amounts)

How to Use
Run Expense_Track.java in a Java IDE or terminal.

Choose from the menu:
Add Expenses
View Expenses
Total Expenses
Highest Expenses
Exit

Enter expense names and amounts when prompted.

Example
Add expense: Food with amount 150
Add expense: Transport with amount 50

View expenses output:
Expense: FOOD | Amount: ₹150
Expense: TRANSPORT | Amount: ₹50

Total expense: ₹200
Highest expense: FOOD | Amount: ₹150
