# N-Queens Problem (Java)

This project solves the N-Queens problem using Backtracking.

## Problem Statement
Place N queens on an NÃ—N chessboard such that no two queens attack each other.

## Concepts Used
- Backtracking
- Recursion
- 2D Arrays
- DSA

## How to Run
1. Compile all Java files
2. Run Main.java
3. Enter value of N
